💡 Idea

Model a car and simulate actions.

📦 Class: Car

Attributes:

brand (String)
model (String)
year (int)
speed (int)
isOn (boolean)

Methods:

turnOn()
turnOff()
accelerate(int value)
brake(int value)
showStatus()

🧪 Expected behavior:
Car is off → cannot accelerate
Turn on → accelerate → brake → show speed

State control (isOn)
