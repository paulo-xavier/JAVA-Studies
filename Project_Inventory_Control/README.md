# 📦 💻 Briefing – Inventory System (Console In Java)

## 🎯 Goal

Develop a system in Java (Console mode) with the function of managing products in inventory. The main functions are: product registration, product exchange and product report.  

O sistema deve funcionar através de um menu interativo, utilizando:
The system must works using a menu with:

- Iteration structures
- Loop structures
- Arrays
- Functions

## Structure

- Product (String)
- Quantity (Int)
- Price (Float)

## Obligatory functionalities

1. Register product
2. List products
3. Search product
4. Stock-in (product)
5. Stock-out (product)
6. Almost out-of-stock products
